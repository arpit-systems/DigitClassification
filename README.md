# DigitClassification

This assignment is based on digit classification using a given Python script from sklearn.

I created a conda environment, installed the required packages and ran the code successfully.

## How to run

1. Create a conda environment
2. Install required packages:
   pip install -r requirements.txt
3. Run the code:
   python plot_digits_classification.py

## Output

The program prints:
- Classification report (accuracy, precision, recall)
- Confusion matrix

The code runs successfully after installing the required dependencies.
